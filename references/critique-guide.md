# 디자인 리뷰 심층 가이드

> Phase 7의 상세 참고 자료입니다. 채점 기준, 장면별 중점 사항, 자주 발생하는 문제 목록을 제공합니다.

## 채점 기준 상세 설명

### 1. 철학 일관성 (Philosophy Alignment)

| 점수 | 기준 |
|------|------|
| 9-10 | 선택한 철학의 핵심 정신을 완벽하게 구현하며, 모든 디테일에 철학적 근거가 있습니다 |
| 7-8 | 전체 방향이 올바르고 핵심 특성이 갖춰져 있으나, 일부 디테일이 벗어나 있습니다 |
| 5-6 | 의도는 느껴지지만 실행 과정에서 다른 스타일 요소가 혼입되어 순도가 낮습니다 |
| 3-4 | 표면적인 모방에 그치며, 철학의 핵심을 이해하지 못했습니다 |
| 1-2 | 선택한 철학과 거의 관련이 없습니다 |

**리뷰 체크포인트**：
- 해당 디자이너 / 기관의 특징적인 기법을 사용했습니까?
- 색상, 폰트, 레이아웃이 해당 철학 체계에 부합합니까?
- 「자기모순적」인 요소가 있습니까? (예：Kenya Hara를 선택했는데 콘텐츠를 가득 채운 경우)

### 2. 시각적 위계 (Visual Hierarchy)

| 점수 | 기준 |
|------|------|
| 9-10 | 사용자의 시선이 자연스럽게 디자이너의 의도대로 흐르며, 정보 파악에 마찰이 없습니다 |
| 7-8 | 주요/보조 관계가 명확하며, 가끔 1-2곳 위계가 모호한 부분이 있습니다 |
| 5-6 | 제목과 본문은 구분되지만 중간 위계가 혼란스럽습니다 |
| 3-4 | 정보가 평평하게 나열되어 있고 명확한 시각적 진입점이 없습니다 |
| 1-2 | 혼란스러워 사용자가 어디를 먼저 봐야 할지 모릅니다 |

**리뷰 체크포인트**：
- 제목과 본문의 폰트 크기 대비가 충분합니까? (최소 2.5배)
- 색상 / 굵기 / 크기로 3-4개의 명확한 위계를 구축했습니까?
- 여백이 시선을 유도하고 있습니까?
- 「눈 가늘게 테스트」：눈을 가늘게 뜨고 봐도 위계가 명확합니까?

### 3. 디테일 완성도 (Craft Quality)

| 점수 | 기준 |
|------|------|
| 9-10 | 픽셀 단위로 정확하며, 정렬 / 간격 / 색상에 어떠한 흠도 없습니다 |
| 7-8 | 전반적으로 정제되어 있고, 1-2곳의 미세한 정렬 / 간격 문제가 있습니다 |
| 5-6 | 기본 정렬은 되어 있으나 간격이 일관되지 않고, 색상 사용이 체계적이지 않습니다 |
| 3-4 | 명확한 정렬 오류, 간격 혼란, 색상 과다 사용 |
| 1-2 | 거칠며 초안처럼 보입니다 |

**리뷰 체크포인트**：
- 통일된 간격 시스템을 사용했습니까? (예：8pt 그리드)
- 동일 유형 요소의 간격이 일관됩니까?
- 색상 수가 적절히 제한되어 있습니까? (일반적으로 3-4가지 이하)
- 폰트 패밀리가 통일되어 있습니까? (일반적으로 2가지 이하)
- 가장자리 정렬이 정확합니까?

### 4. 기능성 (Functionality)

| 점수 | 기준 |
|------|------|
| 9-10 | 모든 디자인 요소가 목표에 기여하며, 불필요한 요소가 없습니다 |
| 7-8 | 기능 지향이 명확하며, 삭제 가능한 장식 요소가 소량 있습니다 |
| 5-6 | 기본적으로 사용 가능하지만, 주의를 분산시키는 장식 요소가 눈에 띕니다 |
| 3-4 | 형식이 기능을 앞서며, 사용자가 정보를 찾기 위해 노력해야 합니다 |
| 1-2 | 장식에 완전히 묻혀 정보 전달 능력을 잃었습니다 |

**리뷰 체크포인트**：
- 어떤 요소 하나를 삭제해도 디자인이 나빠지지 않는다면 삭제해야 합니다.
- CTA / 핵심 정보가 가장 눈에 띄는 위치에 있습니까?
- 「예뻐 보여서 추가한」 요소가 있습니까?
- 정보 밀도와 매체가 일치합니까? (PPT는 너무 밀도 높지 않게, PDF는 더 밀도 있게 가능)

### 5. 창의성 (Originality)

| 점수 | 기준 |
|------|------|
| 9-10 | 신선한 느낌을 주며, 해당 철학 프레임 안에서 독자적인 표현을 찾아냈습니다 |
| 7-8 | 자기만의 아이디어가 있으며, 단순한 템플릿 적용이 아닙니다 |
| 5-6 | 평범하고 템플릿처럼 보입니다 |
| 3-4 | 클리셰를 대거 사용했습니다 (예：AI를 나타내는 그라디언트 구체) |
| 1-2 | 완전히 템플릿이거나 소재를 짜깁기한 수준입니다 |

**리뷰 체크포인트**：
- 흔한 클리셰를 피했습니까? (아래 「자주 발생하는 문제 Top 10」 참조)
- 디자인 철학을 따르면서도 개인적인 표현이 있습니까?
- 「예상치 못했지만 매우 타당한」 디자인 결정이 있습니까?

## 장면별 리뷰 중점 사항

결과물 유형에 따라 리뷰 중점이 다릅니다：

| 장면 | 가장 중요한 항목 | 두 번째 중요 | 완화 가능 |
|------|----------------|------------|----------|
| 카드 / 배너 이미지 | 창의성, 시각적 위계 | 철학 일관성 | 기능성 (단일 이미지는 인터랙션 없음) |
| 인포그래픽 | 기능성, 시각적 위계 | 디테일 완성도 | 창의성 (정확성 우선) |
| PPT / Keynote | 시각적 위계, 기능성 | 디테일 완성도 | 창의성 (명료성 우선) |
| PDF / 백서 | 디테일 완성도, 기능성 | 시각적 위계 | 창의성 (전문성 우선) |
| 랜딩 페이지 / 공식 사이트 | 기능성, 시각적 위계 | 창의성 | — (전방위 요구) |
| 앱 UI | 기능성, 디테일 완성도 | 시각적 위계 | 철학 일관성 (사용성 우선) |
| SNS 이미지 | 창의성, 시각적 위계 | 철학 일관성 | 디테일 완성도 (분위기 우선) |

## 자주 발생하는 디자인 문제 Top 10

### 1. AI 기술 클리셰
**문제**：그라디언트 구체, 디지털 레인, 파란 회로 기판, 로봇 얼굴
**왜 문제인가**：사용자가 이미 이런 시각적 표현에 피로감을 느끼며, 다른 작품과 구별되지 않습니다
**수정**：직접적인 기호 대신 추상적 은유를 사용하세요 (예：채팅 거품 아이콘이 아닌 「대화」의 은유)

### 2. 폰트 크기 위계 부족
**문제**：제목과 본문의 차이가 너무 작음 (<2.5배)
**왜 문제인가**：사용자가 핵심 정보를 빠르게 찾을 수 없습니다
**수정**：제목은 본문의 최소 3배 (예：본문 16px → 제목 48-64px)

### 3. 색상 과다 사용
**문제**：5가지 이상의 색상 사용, 주요/보조 구분 없음
**왜 문제인가**：시각적 혼란, 브랜드 인식 약화
**수정**：주색 1개 + 보조색 1개 + 강조색 1개 + 그레이스케일로 제한하세요

### 4. 간격 불일치
**문제**：요소 간격이 임의적이고 체계가 없음
**왜 문제인가**：비전문적으로 보이며 시각적 리듬이 흐트러집니다
**수정**：8pt 그리드 시스템 구축 (간격은 8/16/24/32/48/64px만 사용)

### 5. 여백 부족
**문제**：모든 공간이 콘텐츠로 채워져 있음
**왜 문제인가**：정보가 빽빽하면 읽기 피로도가 높아지고 오히려 정보 전달 효율이 떨어집니다
**수정**：여백이 전체 면적의 최소 40%를 차지하도록 하세요 (미니멀 스타일은 60%+)

### 6. 폰트 과다 사용
**문제**：3가지 이상의 폰트 사용
**왜 문제인가**：시각적 노이즈, 통일감 약화
**수정**：최대 2가지 폰트 (제목 1개 + 본문 1개), 굵기와 크기로 변화를 만드세요

### 7. 정렬 불일치
**문제**：왼쪽 정렬, 중앙 정렬, 오른쪽 정렬이 혼재
**왜 문제인가**：시각적 질서감을 해칩니다
**수정**：하나의 정렬 방식을 선택하고 (왼쪽 정렬 권장) 전체에 통일하세요

### 8. 장식이 콘텐츠보다 강조됨
**문제**：배경 패턴 / 그라디언트 / 그림자가 주요 콘텐츠를 가림
**왜 문제인가**：주객이 전도되어, 사용자는 장식이 아닌 정보를 보러 왔습니다
**수정**：「이 장식을 삭제하면 디자인이 나빠지는가?」 아니라면 삭제하세요

### 9. 사이버 네온 남용
**문제**：짙은 파란 배경(#0D1117) + 네온 색상 발광 효과
**왜 문제인가**：기본 취향의 금기 구역이며 (본 스킬의 취향 기준선), 가장 큰 클리셰 중 하나가 되었습니다 — 자신의 브랜드에 맞게 override 가능합니다
**수정**：더 식별력 있는 색상 방안을 선택하세요 (20가지 스타일의 색상 시스템 참조)

### 10. 정보 밀도와 매체 불일치
**문제**：PPT에 한 페이지 가득 텍스트 / 커버 이미지에 10개 요소를 채워 넣음
**왜 문제인가**：매체마다 최적의 정보 밀도가 다릅니다
**수정**：
- PPT：페이지당 핵심 메시지 1개
- 커버 이미지：시각적 초점 1개
- 인포그래픽：계층적으로 표시
- PDF：더 밀도 있게 가능하지만 명확한 내비게이션 필요
